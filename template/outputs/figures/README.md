# Research Visualizations & Figures

## 📊 Overview
This directory contains all graphical outputs generated by the analysis pipeline. Following **FAIR4RS** and **Open Science** guidelines, these figures are reproducible and linked to specific versions of the code and data.

---

## ⚡ Reproducibility Rule
All images in this folder are **automatically generated**. 
*   **Source Script:** `code/04_visualization.R` (or similar).
*   **Manual Edits:** Do **not** manually edit these files in Photoshop/Illustrator. Any aesthetic changes must be implemented in the R code to ensure the "Provenance" of the scientific result is preserved.

---

## 📂 Figure Metadata & Formats

To ensure **Interoperability** and high-quality publication standards, figures are saved in the following formats:

| Format | Purpose | Feature |
| :--- | :--- | :--- |
| `.png` | Web / GitHub Preview | High-resolution raster (300 DPI). |
| `.pdf` / `.svg` | Publication / Print | Vector-based for infinite scalability. |
| `.eps` | Journal Submission | Required by some legacy publishers. |

### Variable Definitions
Variable names and scales used in these figures correspond directly to the **Data Dictionary** found in `docs/` or `data/processed/`.
